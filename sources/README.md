# sources/

Informational sites used for scoring and specs — same conventions as
`../stores/` (`package.d` access layer, `README.md` for quirks, either may
be absent), but nothing is bought here, so store-specific knowledge
(stock, shipping, payment) doesn't apply.

| Directory | Site | Provides |
|-----------|------|----------|
| `cpubenchmark/` | cpubenchmark.net | CPU PassMark scores (used by server purchases) |
| `sammobile/` | sammobile.com | Samsung device spec/firmware DB |
